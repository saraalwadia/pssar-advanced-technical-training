## Research Trail Project

### Overview
This project is part of a technical training program in Data Handling and Analysis.  
It demonstrates how to work with real-world datasets using Python and Pandas.

### Objective
The goal is to:
- Load and explore multiple data formats (CSV, JSON, Excel)
- Clean and filter data
- Extract insights using aggregation and analysis
- Connect insights across multiple datasets

---

### Project Structure

raw_data/
- researchers.csv
- publications.json
- funding.xlsx

scripts/
- research_trail.py

---

## Tasks

### CP1 - Researchers Analysis
- Load researchers dataset
- Filter active researchers with h_index > 15
- Sort by joined_year
- Extract first letters of last names to form a hidden word

### CP2 - Publications Analysis
- Load JSON data
- Find the most cited paper
- Identify the related researcher

### CP3 - Funding Analysis
- Load Excel file
- Clean dirty financial data
- Calculate total funding amount
